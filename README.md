# Weather JS App

## Overview
A simple weather application that fetches live data from OpenWeatherMap API using JavaScript. Demonstrates asynchronous programming, API consumption, and basic error handling.

## Features
- Search for city weather by name
- Display temperature, weather description, and city info
- Error handling for invalid city names
- Uses async/await and Fetch API

## Tech Stack
- JavaScript (ES6)
- HTML, CSS
- Fetch API, Async/Await

## How to Use
1. Open `index.html` in a browser
2. Enter city name and click the search button
3. Weather info will display; errors will show if city not foun
